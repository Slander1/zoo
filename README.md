В прошлой компании для прототипов игр использовалась собстсенная реализация Event Bus.
Для полноценных игр использовался Extenject.

В данном тестовом решил поработать с новыми технологиями и использовать VContiner. 
С ним последний раз сталкивался около 3-4 лет назад.

Инициадизация и запуск проекта происходит через VContainer
Project root LifetimeScope , отсюда убирается необходимость Initialize сцены

SceneCoordinator сделан по простейшей логике , 
задание не подрузамевает его в принципе, но он является неотеъмлемой частью любой игры, если мы захотим ее немного дополнить.
К тому же загрузка через Init сцену всегда ускоряет холодный или горячий старт приложения.
